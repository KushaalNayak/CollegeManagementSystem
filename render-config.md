# Render Configuration

## Backend Deployment
- **Build Command:** `mvn clean install`
- **Start Command:** `java -jar target/college-management-system-1.0.0.jar`
- **Environment:** Java 17

## Frontend Deployment
- **Build Command:** `npm run build`
- **Publish Directory:** `build`
- **Environment:** Node.js 18

## Database
- **Type:** PostgreSQL
- **Plan:** Free tier
